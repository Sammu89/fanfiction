## The 3 states

### Hide

Use when you just want content off the public site for now.

* Readers can’t see it.
* The author can still edit it and change it back.

Works for stories and chapters.

### Block

Use when moderators needs to freeze content.

* Readers lose access.
* The author can still open it, but **view-only** (no editing or delete).
* Moderators can access it normally.

Works for stories and chapters.

### Suspend user

Use when the restriction must apply to the whole user account.

* The user can log in and view their dashboard.
* They can’t create/edit anything until unsuspended.
* Cant comment on any story
* Their stories become **blocked** (so readers lose access).
* Moderators can still unblock a specific story if they intentionally want it public while the user remains suspended.

### Summary

* **Hide** = not public, but the author can still edit it
* **Block** = moderator lock: not public + author can view it but no longer delete or edit it
* **Suspend** = account ban: user becomes view-only + their stories are all blocked

---

Moderators can:

* Make stories or chapters visible / hide
* Block / unblock
* Edit / delete

When a story or chapter is **blocked**:

* It is removed from public access (even if it was visible before).
* The author can open it, but only in **view-only** mode.

---

## User account

Moderators can:

* Suspend / unsuspend

When a user is **suspended**:

* Their stories become **blocked** (public loses access).
* The user becomes read-only until unsuspended.

---

## What the author sees

### If their story is blocked

They can open the story and its chapters, but cannot:

* Edit the story
* Add chapters
* Edit blocked chapters
* Delete the blocked story

### If one chapter is blocked

It still appears in their chapter list. They can open it, but cannot:

* Edit it
* Delete it
* Unblock it

### If their account is suspended

They can log in and view their content, but cannot:

* Create/edit stories or chapters
* Use normal author management features
* Comment as an active user

---

## What readers see

If something is **hidden** or **blocked**, readers completely loose access to it.

---

## Quick summary

| Situation       | Admins / Mods               | Author                                       | Regular Readers                                        |
| --------------- | --------------------------- | -------------------------------------------- | ------------------------------------------------------ |
| Story hidden    | Can edit, show/hide, delete | Can manage normally unless blocked/suspended | No access                                              |
| Story blocked   | Can edit, unblock, delete   | View-only                                    | No access                                              |
| Chapter hidden  | Can edit, show/hide, delete | Can manage normally unless blocked/suspended | No access                                              |
| Chapter blocked | Can edit, unblock, delete   | View-only                                    | No access                                              |
| User suspended  | Can unsuspend + moderate    | View-only, no author actions                 | Can’t interact with their now-blocked stories normally |
