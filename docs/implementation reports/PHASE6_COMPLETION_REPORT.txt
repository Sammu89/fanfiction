================================================================================
                        QUICK START GUIDE
================================================================================

FOR AUTHORS:
1. Log in to WordPress
2. Navigate to /plugin_base_name/dashboard_custom_name/
3. Click "Create New Story"
4. Fill in title, introduction, genres, and status
5. Click "Create Story"
6. Click "Add Chapter" to add content
7. Click "Edit Profile" to update bio/avatar

FOR ADMINISTRATORS:
1. Stories appear in Stories admin list
2. Can edit/delete via admin interface
3. Configure dashboard URLs in Settings > URL Name Rules
4. View moderation queue for reports

FOR DEVELOPERS:
- Check includes/class-fanfic-author-dashboard.php for handlers

================================================================================
                      SUPPORT & TROUBLESHOOTING
================================================================================

If you encounter issues:

1. Check file permissions (should be readable by web server)
2. Verify WordPress rewrite rules (flush in Settings > Permalinks)
3. Check user roles (author should have fanfiction_author role)
4. Enable WP_DEBUG in wp-config.php
5. Check wp-content/debug.log for errors
6. Verify browser console for JavaScript errors

For detailed troubleshooting, see PHASE6_IMPLEMENTATION_SUMMARY.md
